# yatube_project
### Description
Social Network for bloggers.
### Stack
Django  2.2.19
pytz  2022.6
sqlparse  0.4.3

### Launch in Dev mode
- Install and run virtual environment
- Install dependencies with requirements.txt
```
pip install -r requirements.txt
``` 
- In the folder with manage.py run:
```
python3 manage.py runserver
```
